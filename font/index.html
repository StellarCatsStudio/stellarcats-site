導入手順

  1. このページと同じフォルダに StellarCatsfont_v1.0.woff2 を置く。
  2. 上の @font-face のファイル名を実物に合わせる。
  3. 必要に応じて .otf も配布（Webは .woff2 推奨）。

用語：@font-face は“フォントを読み込むCSSの書き方”。

注意点（最小）

  • Windows/Mac/主要アプリで表示確認
  • 収録文字は README と一致させる
  • デモ版と製品版はファイル名で明確化
